# Silent Sanctuary - Privacy Policy

## 1. Introduction

Silent Sanctuary is committed to protecting your privacy. This Privacy Policy explains how we collect, use, disclose, and safeguard your information when you use our platform.

## 2. Information We Collect

### Essential Information
- Email address (for account recovery and notifications)
- User ID (6-digit identifier for login)
- Password hash (never stored in plaintext)
- Vault PIN hash (for chat access)
- Account creation date

### Non-Essential Information We Don't Collect
- Browsing history
- Location data
- Device fingerprints
- Tracking cookies
- Message content (encrypted end-to-end)
- IP addresses (except for security logging)

### Automatically Collected Information
- Login timestamps
- Failed login attempts
- Device verification events
- Security alerts

## 3. How We Use Your Information

### Primary Uses
- Account authentication and security
- Sending security notifications
- Preventing fraud and abuse
- Complying with legal obligations

### Secondary Uses
- Improving platform security
- Analyzing usage patterns (anonymized)
- Sending important updates
- Responding to support requests

### What We Don't Do
- Sell your data to third parties
- Use your data for marketing
- Share your data with advertisers
- Create user profiles for targeting
- Use cookies for tracking

## 4. Data Retention

### Message Data
- Deleted from memory when you log out
- Not stored on servers
- Not stored in browser
- Permanent deletion on session end

### Account Data
- Retained until account deletion
- Can be deleted anytime
- Automatic deletion after inactivity (optional)
- Backup retention: 30 days

### Security Logs
- Login attempts: 30 days
- Failed authentications: 30 days
- Security events: 90 days
- Audit logs: 1 year (encrypted)

### Email Notifications
- Delivery logs: 7 days
- Bounce records: 30 days
- No content retention
